+====================
+config
+====================
+
+A repository for my dotfiles and other assorted scripts.
+
+--------
+Contents
+--------
+
+sh - various shell configurations, shortcuts + other niceties.
+	inputrc - bash/python shell vim mode + other
+	bashrc 	- bash configuration
+	zshrc	- zsh configuration
+	profile - contains shortcuts that are global to all configs.
+
+
+resources - document templates, desktop backgrounds
+
+vim - vimrc and ftplugins.
+	vimrc 	- global configurations for vim. Highlights include laststatus, vundle and relative numbering.	
+	ftplugins/tex	- filetype plugin for LaTeX. Still under development but I hope to add a notes mode.
+
